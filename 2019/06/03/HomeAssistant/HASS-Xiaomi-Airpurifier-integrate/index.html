发表于 2019-06-03 更新于 2026-01-09 分类于 HomeAssistant 阅读次数： Disqus：

HA Release: 0.94.4

净化器型号: AC-M2-AA

Model info: zhimi.airpurifier.m2

4. 下载 packages 文件

下载 xiaomi_air_pufifier.yaml 放入 ~/homeassistant/packages 目录下，此文件修复了上面参考链接里的bug和 HA 升级导致的几个问题，主要为：

  • 增加 sensors: xiaomi_ap_filter_used 和 xiaomi_ap_filter_life，以在主界面显示滤芯使用时长和滤芯寿命
  • 修复提示音无效（HA 升级，service 名称改变导致）

注意：需要替换文件中的 host 和 token ！！！
